Mobile Touch Interface Test Instructions

How to test:

  1. Open this page on a mobile device or use browser dev tools with mobile simulation
  2. Tap each button to ensure it responds properly
  3. Check that all buttons meet the minimum 44x44px touch target (red dashed outline)
  4. Verify touch feedback works (buttons should provide visual feedback on tap)
  5. Test the hamburger menu opens/closes properly
  6. Ensure form inputs don't zoom on iOS (font-size ≥ 16px)

Pass criteria: All buttons should be easily tappable, provide feedback, and meet accessibility standards.
