🔍 Logo Issue Analysis Results

❌ ISSUE #1: HTML Structure Problem

Problem: Both <img> and <span> are inside the same <a> tag

Effect: When image fails and gets display:none, the span is still inside the flexbox layout which can cause positioning issues

Current Structure (PROBLEMATIC):

<a href="index.html" class="apple-nav-logo">
  <img src="..." id="header-logo" onerror="...">
  <span id="text-logo-fallback" style="display: none;">Scoop Unit 🐾</span>
</a>

❌ ISSUE #2: CSS Flexbox Conflict

Problem: .apple-nav-logo has `display: flex !important`

Effect: The flex container affects how child elements (img and span) are laid out, potentially causing the span to not display correctly even when shown

❌ ISSUE #3: JavaScript Scope Problem

Problem: Multiple scripts trying to handle the same logo with different approaches

Effect: Preloader script, visibility script, and inline onerror might conflict
